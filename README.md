# shopify-stock-dashboard-analysis
Interactive Excel dashboard project analyzing Shopify stock performance, volatility, moving averages, and trading volume trends.
# Shopify Stock Dashboard Analysis

## Project Overview
This project is an interactive Excel dashboard analyzing Shopify stock performance from 2015 to 2025.

## Key Features
- Stock price trend analysis
- 30-day moving average
- Volatility analysis using standard deviation
- Trading volume trend analysis
- Interactive slicers and KPI cards

## Key Insights
- Highest growth year: 2021 (23%)
- Highest volatility: 2022
- Strong bullish momentum in 2020–2021

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Statistical analysis
